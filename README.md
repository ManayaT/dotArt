# dotArt

## プロジェクトの概要
- プロジェクトの目的<br>
　i.　ゼミで学習した[幅優先探索][A*アルゴリズム]のアウトプット<br>
　ii.　ドット絵を描くことができるアプリケーションの作成

## システム概要
　このWebアプリケーションは，以下の機能を実装している<br>
　　i.　点の描画<br>
　　ii.　直線の描画<br>
　　iii.　エリア内の塗りつぶし<br>
　　iV.　A*アルゴリズムの描画<br>
　　V.　キャンバスサイズの変更<br>
　　Vi.　パレットへの色追加<br>
　　Vii.　パレット上の選択色の削除<br>
　　Viii.　キャンバスのクリア<br>
　　iX.　png画像への変換（Chromeのみ）<br>
